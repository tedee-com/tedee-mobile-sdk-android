class LockConnectionManager(context: Context) : ILockInteractor

Lock connection manager. Manages connection with Tedee lock. To establish successful connection with lock you need to provide serial number, device certificate and mobile public key.

open override fun setSignedTime(signedTime: SignedTime)

Sends the SET_SIGNED_TIME (0x71) command to the lock with the provided signed time. The lock's response will be communicated through the ILockConnectionListener.onIndication method. This command is crucial for ensuring that the lock's operations are synchronized with a trusted time source, enhancing the security and reliability of its operations.
